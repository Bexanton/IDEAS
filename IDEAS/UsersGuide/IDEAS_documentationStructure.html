Structure of IDEAS documentation:

General description

Goal

Short description of goals of model

Description

Model description: components, equations, reference links (references should be written in the user's Guide of the library)

Assumptions and limitations

(keep this section short and to the point, preferentially in bullets only)

  1. Description of assumptions used by the model.
  2. Description of the validity region of the model and interval of accuracy (if possible)

Parameters

  1. Which are the important, the tuning and the final parameters (that should be tuned).
  2. What are the default value of the model and how have then been chosen.

Implementation (Optional)

Explain implementation of model

Validation

Description of the validation procedure and results

Example (optional)

Link to an example
